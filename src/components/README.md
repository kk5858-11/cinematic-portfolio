# components/

展示层组件，按职责分层。

| 目录 | 职责 |
|------|------|
| `ui/` | shadcn 原子组件 |
| `layout/` | 全局布局壳 |
| `sections/` | 首页各区块（组合 animations + webgl） |
| `blog/` | MDX 映射与列表 UI |
| `providers/` | Context Providers |
| `seo/` | 结构化数据组件 |
| `interactive/` | 磁吸、Tilt 等交互壳 |

动画实现 → `@/animations` · 3D → `@/webgl`
